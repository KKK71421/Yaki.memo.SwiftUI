# キッチン手順確認＆シフト管理アプリ (SwiftUI版)

アルバイト先の居酒屋におけるキッチン業務の効率化と、新人教育のスピード向上を目的として制作したiOSネイティブアプリです。
元々HTML/JavaScriptで運用していたWebアプリをベースに、よりスマートフォンでの操作性に特化したUI/UXを提供するため、SwiftUIを用いて再設計・実装しました。

## 🛠 解決した課題と機能
- **ポジション別の手順確認（アコーディオンUI）**: 
  刺し場・揚げ場・焼き場ごとの調理手順や味付けを、現場のスピードに合わせてその場で素早く展開・確認できる「ひょこっ」とした開閉UI（`DisclosureGroup`）を実装。
- **ポーション（定量）の可視化**: 
  ミスが起きやすい仕込み・ポーション量を一覧で即座に確認可能。
- **自動更新対応のシフト表ビュー**: 
  Googleドライブの画像URLから自動でサムネイルIDを抽出し、アプリ内で非同期（`AsyncImage`）に最新のシフト表を表示。
- **セキュリティ（合言葉画面）**: 
  店舗内の機密情報（シフトURLやパスワード）を守るため、起動時にロック画面を挟む設計。誤入力時にはアニメーションによるフィードバックを実施。

## 💻 技術スタック
- **言語**: Swift 5.10
- **フレームワーク**: SwiftUI
- **開発環境**: Xcode
- **非同期処理**: AsyncImage (Google Drive API Thumbnailを活用)

## 📦 コードについて
本リポジトリ内の `ContentView.swift` にフロントエンドのロジックおよびメニューデータをすべて集約して管理しています。

*※現在、実際の店舗で運用しているPrivateリポジトリから、セキュリティに配慮して機密情報をダミーに置き換えたコードをポートフォリオ用に公開しています。*
